This law states that left multiplication by any given element squares to some constant map.

The number of magmas of size n satisfying this law grows as `exp(n^2 log(n) + O(n^2 log(log(n))))`, more precisely the same as for [law 9](https://teorth.github.io/equational_theories/implications/?9), up to an `exp[O(n log(n))]` factor.
